# MissingCash — LIVE site backup

Full offline backup of the live site at **https://missingcash.jmorganegypt.workers.dev/**,
captured 2026-06-15. This is the version currently serving (a built React/Vite SPA),
saved before any DNS/deploy changes could disturb it.

## What this is
A compiled **single-page app** — the whole site lives in the JS + CSS bundle, not in
separate HTML files. Title: "MissingCash — Find Your Unclaimed Money in Australia".

Client-side routes (all rendered from the bundle, no separate files):
`/` · `/crypto` · `/finance` · `/guides` · `/contact` · `/privacy`
